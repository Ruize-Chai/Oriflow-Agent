# Frontend (Vue + TypeScript + Axios)

This is a minimal Vite + Vue3 + TypeScript frontend that demonstrates the Oriflow backend APIs.

Quick start

```bash
cd Front
npm install
npm run dev
```

The app expects the backend API at `http://localhost:8000` by default. To change base URL, set `VITE_API_BASE` in the environment.
# Oriflow Front (Vite + Vue3 + TypeScript)

Quick start (frontend):

```bash
cd Front
npm install
npm run dev
```

The frontend expects the backend API at `http://localhost:8000` by default. To change, set `VITE_API_BASE` env var when running Vite.
